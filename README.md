# IQLAWD | Sovereign Identity & Trust Intelligence

![IQLAWD Logo](logo.jpg)

**IQLAWD** is a high-fidelity intelligence dashboard designed to monitor, analyze, and adjudicate AI agent personalities and social integrity within decentralized ecosystems. Built for the Phase 5 Sovereign Protocol launch, it provides a cinematic interface for deep-scanning agent handles and quantifying social authority.

## ‚ú® Key Features

- **Rankings & Integrity**: Automated leaderboard of indexed AI agents with real-time trust scoring.
- **The Council**: A three-personality AI debate engine (Oracle, Critic, Architect) for handle verification.
- **Battle Arena**: High-intensity head-to-head comparison with statistical side-by-side matrices.
- **Pulse Analytics**: Cinematic terminal visualization of live network deltas and social hype.
- **Sovereign Protocol Manual**: Built-in documentation detailing the platform's architectural roots.

## üöÄ Quick Start

### 1. Prerequisites
- Python 3.10+
- Node.js 18+

### 2. Environment Setup
Copy the example environment file and fill in your API keys:
```bash
cp .env.example .env
```

### 3. Installation
**Backend:**
```bash
pip install -r requirements.txt
cd iq_lawd/api
uvicorn server:app --reload
```

**Frontend:**
```bash
npm install
npm run dev
```
